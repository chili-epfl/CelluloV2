class Cellulo::CelluloZone

CelluloZone Base Class for zones.

Abstract description of a "zone" on a 2D plane. These zones could be closed curves, polygons, open curves, even points.

They are meant interact with CelluloZoneClients (such as a robot, or a virtual robot on a screen), calculating a real value with respect to each client (for example, the distance to the client in the case of a point zone). With the help of CelluloZoneEngine, a zone emits each client's zoneValueChanged() signal upon the changing of the value with respect to that client.
