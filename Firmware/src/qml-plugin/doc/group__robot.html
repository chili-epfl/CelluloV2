◆ nextGoalPoseVelRequested

void Cellulo::CelluloRobot::nextGoalPoseVelRequested ( qreal  deltaTime)
signal

Emitted when the controller needs the next goal pose and velocity.

Upon receiving this signal, setGoalPoseAndVelocity() should be called by the user if a user control loop that cycles on each received pose of the robot is present.

Parameters
deltaTimeDelta time in milliseconds that should be used if a controller differentiates or integrates in a loop triggered by this signal
